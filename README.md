# Портфолио лабораторных работ

Данный репозиторий содержит исходные файлы и собранный статический сайт, созданный с помощью MkDocs.

## Выбор темы

Для оформления сайта выбрана тема **Material for MkDocs**, так как она:
- обеспечивает удобную навигацию;
- имеет современный внешний вид;
- хорошо подходит для документации и учебного портфолио;
- поддерживает аккуратную иерархию разделов.
